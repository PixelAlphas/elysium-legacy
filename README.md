# elysium-legacy
A archive of elysium map created for BAO's 8K Jam event. - Will not be receive any fixes or update.

Game version requirement - Bedrock 1.19.20 only.

# Original Description
- Elysium -

A empty void, abandoned place where you awaken beneath of witch towner, with alone robot that guides you wherever you should go.

You look abandoned buildings for resources gear up and get yourself grappling hooks.

---

This map is short'est'-storyline based rpg makers game stuff, with a small mechanic like grappling hook, quests systems, animated textboxs etc for this demo.

Grappling hooks acting wonky after using it too much, Some dialogue does not work well with text anim disabler ( usually I fix by rewriting stuff but honestly I have no time. ), and some translation, overall it shouldn't bother that much, it wasn't ground breaking bugs, rather just visuals.
